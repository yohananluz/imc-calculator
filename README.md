# 🧮 Calculadora de IMC em Python

Este é um projeto desenvolvido em Python para calcular o Índice de Massa Corporal (IMC). O objetivo é aplicar conceitos de lógica de programação, tratamento de erros e estruturas condicionais.

## 🚀 Funcionalidades
- Entrada de peso (kg) e altura (m).
- Cálculo automático do IMC.
- Classificação do resultado baseada na tabela da OMS (Abaixo do peso, Peso normal, Sobrepeso, Obesidade).
- Tratamento de exceções para entradas inválidas (letras ou números negativos).

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Ambiente:** Terminal / Console
